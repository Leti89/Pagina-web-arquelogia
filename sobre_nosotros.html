Hola! Somos el grupo 2do ME de la Escuela Tecninca El Pinar.

Encontramos un problema en relacion a descubrimientos historicos precolombinos del Uruguay, estos estan dispersos al rededor del internet y pensamos que estaria bueno si toda la informacion estuviera mas organizada.

Se quiere aplicar la tecnologia a este problema e incluir robotica terrestre, exploracion aerea con dron, desarollo web y videojuegos educativos para simular la busqueda y recuperacion de objetos historicos del pasado, promoviendo la valoracion del patrimonio cultural local de forma accesible, ludica e innovadora.

El sistema se compone de:

1) Exploracion aerea con dron:

  • El dron sobrevuela el área de exploración simulada y capta imágenes y video del terreno.
  • Con base en estos datos, se delimita la zona segura y se marca la ruta más eficiente para el desplazamiento del robot terrestre.
  • Esta información se utiliza tanto para la planificación de la exploración, como para alimentar el contenido multimedia de la web y el videojuego.

2) Robot recolector de objetos

  • Diseñado para recorrer el terreno explorado previamente por el dron.
  • Recolecta objetos que simulan “tesoros del pasado” (pueden ser impresiones 3D de herramientas, cerámicas, fósiles, etc.).
  • Lleva los objetos hacia una base para ser registrados y digitalizados.
  • Puede contar con sensores para evitar obstáculos y seguir rutas marcadas.

    • 3) Pagina web del proyecto

  • Actúa como bitácora del proceso, donde se documentan:
    • Las misiones del robot y del dron.
    • Las historias detrás de cada objeto.
    • Videos, fotos y entrevistas con vecinos, docentes o historiadores.
  • El sitio será accesible y visualmente amigable para estudiantes de primaria y secundaria.

    • 4) Videojuego educativo

  • Diseño para niños y adolescentes.
  • Recrea la experiencia de exploración: primero se sobrevuela el terreno con el dron, luego se guía al robot para encontrar los objetos.
  • Cada objeto recolectado se desbloquea con información histórica, curiosidades y minijuegos relacionados.
  • Fomenta la investigación, la observación y el aprendizaje lúdico.
