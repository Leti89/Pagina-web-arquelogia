Hola. Somos el grupo 2° ME de la Escuela Técnica El Pinar.

Encontramos un problema en relación a descubrimientos históricos precolombinos del Uruguay. Están dispersos en internet y pensamos que sería mejor si toda la información estuviera organizada.

Se quiere aplicar tecnología a este problema e incluir robótica terrestre, exploración aérea con dron, desarrollo web y videojuegos educativos para simular la búsqueda y recuperación de objetos históricos del pasado.

El sistema se compone de:

1) Exploración aérea con dron

  • El dron sobrevuela el área de exploración simulada y capta imágenes y video del terreno.
  • Con base en estos datos se delimita la zona segura y se marca la ruta eficiente para el robot terrestre.
  • La información se usa para la planificación y para los recursos multimedia de la web y el videojuego.

2) Robot recolector de objetos

  • Recorre el terreno explorado previamente por el dron.
  • Recolecta objetos que simulan tesoros del pasado.
  • Transporta los objetos a una base para registro y digitalización.
  • Puede incluir sensores para evitar obstáculos y seguir rutas.

3) Página web del proyecto

  • Actúa como bitácora donde se documentan:
    • Misiones del robot y del dron.
    • Historias detrás de cada objeto.
    • Videos, fotos y entrevistas.
  • Diseñada para estudiantes de primaria y secundaria.

4) Videojuego educativo

  • Apunta a niños y adolescentes.
  • Recrea la exploración: vuelo con dron y guía del robot.
  • Cada objeto recolectado desbloquea información y minijuegos.
  • Promueve investigación y aprendizaje lúdico.
